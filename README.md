# Union Shop Website 

The union shop website is designed to be a replication the existing UPSU website, that features University branded merchandise, due to copyrighting this website will not use any merchandise related to or imitating that of the university itself and instead will have AI generated images designed to represent the relevant products included. 


## Features 
- **Static Home Page**: Contains a main home page with widgets that load static content
- **Static Nav Bar**: At time of writing, the project contains a navigation bar that is mostly static but can load the SALE and ABOUT US page 
-**About us page**: Fully functional about us page that contains title and description
-**Footer**: Displayed footer that shows the sites title and powered by shopify as well as social media links, refreshes the page when clicked. 
-**Collection Page**: At time of writing one display page added for the navy hoodie, in this page users can adjust size and colour as well as quantity. 
-**Functioning Hero Section**: Displayed Hero Section where users can press arrows to navigate through relevant tabs
